## Run Both Client and Server

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start development servers**:
   ```bash
   npm run dev
   ```

---

## Required Packages

Make sure the following packages are installed:

### Server-side
```bash
npm install express cors mysql2
```

### Client-side (Next.js app)
```bash
npm install axios next
```
